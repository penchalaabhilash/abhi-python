Title: Python Setup Verifier & Hello Utility


Concepts Demonstrated: Introduction to Python, Installation & Setup, Python Syntax, Variables, Input & Output

Description:
This project is a good starting point because the first practical problem every beginner faces is making sure Python is installed correctly and that simple scripts actually run. Instead of only printing "Hello, World!", this utility becomes a small real-world setup checker that confirms Python is working, collects the user’s name, and displays system-related details in a friendly format. It applies Python syntax, variables, and basic input/output in the simplest useful way, while also helping the learner verify that their environment is ready for future projects.
At the end, the program greets the user and confirms the Python installation is working.

Implementation Steps:

Install Python from the official website.
Install a code editor such as VS Code.
Create a file named setup_check.py.
Print a welcome message to confirm the script runs.
Ask the user for their name using input().
Store the name in a variable.
Display a personalized greeting using Python string formatting.
Print the current Python version using the sys module.
Add a small message like “Environment Ready”.
