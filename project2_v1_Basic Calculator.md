-----------------
Basic Calculator:
-----------------

Concepts Demonstrated:
Variables, Data Types, Type Casting, Operators, Input & Output, Control Statements (if, elif, else), Python Syntax

Description:
This smaller version is practical because a calculator is one of the simplest real tools a beginner can build and use immediately. It applies input/output by asking the user for an operation and two numbers, variables to store values, type casting to convert text input into numbers, operators to perform arithmetic, and control statements to choose the correct calculation.
At the end, the program should perform one selected calculation and display the result clearly.



Example Output:
---------------

=== Basic Calculator ===
1. Add
2. Subtract
3. Multiply
4. Divide

Choose an option: 1
Enter first number: 12
Enter second number: 8
Result: 20.0
